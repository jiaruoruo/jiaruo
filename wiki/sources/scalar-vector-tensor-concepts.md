# 标量、矢量(向量)、张量（tensors）的理解

## Summary

博客园技术文章，用通俗语言解释标量、矢量（向量）、张量三个基础数学概念及其层级关系。张量是矢量概念的推广，矢量是一阶张量，标量（零阶张量）和矩阵（二阶张量）都是特殊类型的张量。

## Key Points

- **标量（Scalar）**：只有大小、没有方向的量
  - 例：质量、密度、温度、功、能量、路程、速率、体积、时间、热量、电阻、功率、势能
  - 特性：无论选取何种坐标系，标量数值**恒保持不变**
- **矢量 / 向量（Vector）**：同时具有**大小（magnitude）**和**方向**的量
  - 例：位移、速度、力、加速度、电场强度
- **张量（Tensor）**：矢量概念的推广，矢量是一阶张量
  - 定义：可用来表示矢量、标量和其他张量之间线性关系的**多线性函数**
  - 直观理解：一个 n 维数值阵列
  - 用"阶（rank）"描述维度单位：
    - 零阶张量 = 标量（一个数）
    - 一阶张量 = 向量（一维数组）
    - 二阶张量 = 矩阵（二维数组）
    - 三阶及以上张量 = 多维数组
  - **标量、向量、矩阵等都是特殊类型的张量**

## Quotes

> "张量概念是矢量概念的推广，矢量是一阶张量。张量是一个可用来表示在一些矢量、标量和其他张量之间的线性关系的多线性函数。"

> "每个张量的维度单位用阶来描述，零阶张量是一个标量，一阶张量是一个向量，二阶张量是一个矩阵"
