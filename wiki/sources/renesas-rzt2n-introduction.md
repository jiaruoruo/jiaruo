# RZ/T2N Product Introduction

## Summary

瑞萨 RZ/T2N 产品介绍（2024年，37页，CONFIDENTIAL，产品开发中）。RZ/T2N 是 RZ/T2H 的精简版，同样为异构 SoC，支持4轴电机控制和多协议工业以太网，面向 PLC、伺服驱动器、工业机器人和网络卡应用。

## Key Points

- **RZ/T2N 定位**：RZ/T2H 精简版；4轴电机控制（相比 T2H 的9轴）；多协议工业以太网主/从站；目标应用：PLC、伺服驱动器、工业机器人控制器、工业网络卡
- **RZ/T 产品路线图**：T1（CR4+CM3，600+150MHz）→ T2M（CR52×2，800MHz）→ T2L（CR52，800MHz）→ T2H（CA55×4+CR52×2）→ T2N（CA55+CR52，开发中）
- **与 T2H 区别**：T2N 降低 CPU 规格（单 A55 核或双核，规格开发中），保留多协议网络和多轴电机控制，降低成本，面向中端工业机器人
