# RH850/U2 Series U2BX Microcontroller Introduction

## Summary

瑞萨 RH850/U2B 车规 MCU 产品介绍（Mar-2025，REV 1.19，149页，RENESAS CONFIDENTIAL + NDA）。RH850 是瑞萨汽车 MCU 旗舰系列，28nm 工艺，双 CPU 架构，面向 Zone ECU、自动驾驶、功能安全等汽车应用。累计出货量超7亿颗，失效率<1ppm。

## Key Points

- **产品定位**：汽车 Zone ECU/多核高性能 MCU；28nm 双 Fab（TSMC台湾 + JASM熊本）；强汽车市场积累
- **架构特点**：双 CPU 路线（Renesas 自研 32bit + ARM MCU），可与 R-Car SoC 配合扩展
- **质量记录**：出货量超7亿颗，失效率<1ppm（行业领先水平）
- **与机器人关联**：功能安全（FuSa）技术可迁移至机器人安全关键场景；多核实时控制架构与机器人控制器类似；但本文档主要面向汽车市场
- **文档性质**：CONFIDENTIAL + NDA，内容不对外公开，知识库仅记录公开可知的基本产品定位信息
