# 机器人应用推广指南

## Summary

瑞萨电子中国嵌入式处理器事业部2025年4月发布的机器人应用市场综合指南（65页），涵盖工业机器人、人形机器人、服务机器人三大品类的市场概况、系统架构和瑞萨产品方案，以及未来市场展望。

## Key Points

- **中国工业机器人市场**：中国占全球年安装量51%；2025年工业机器人密度预计500台/万人；主要用于搬运（约50%）和焊接；本土制造商国内市场份额大幅增长
- **机器人分类**：工业机器人（搬运/码垛/包装/喷涂/切割/焊接）+ 服务机器人（家庭/医疗/公共服务）+ 特种机器人（军事/极限作业/应急救援）
- **中国服务机器人市场**：持续高速增长（至2025E约1100亿元）；中国持有机器人相关有效专利超19万项，占全球三分之二
- **瑞萨核心产品定位**：
  - 伺服驱动器（机械臂）：RZ/T2H/T2M/T2L/T2N/N2L/RA8T2/RX72M
  - 移动机器人：RZ/V（替换X86主控）+ RZ/T,N（伺服）
  - 人形机器人：关节（RA8T2/RZ/T2L）+ 关节控制器（RZ/V2H）+ 灵巧手（RA8T2主+RA6/RA4从）
- **功能安全（FuSa）**：STO 功能（Safe Torque Off），RZ/T2M 双核交叉互检+RX72N，IEC 61508 合规
