# GC Humanoid Robot Sharing - Infineon Jun 2025

## Summary

英飞凌（Infineon）Luke Zhang 于2025年6月的中国大陆（GC）人形机器人市场分析与产品方案分享（36页）。聚焦人形机器人市场预测、中国市场格局、关节模块技术挑战，以及英飞凌在电机控制、传感、电源管理等方面的产品定位。

## Key Points

- **市场预测**（Goldman Sachs）：2025年人形机器人全球出货量>10,000台；2030-2031年达100万台里程碑；2025-2035 CAGR 基础情景>50%，乐观情景>70%
- **中国市场**：2025年中国出货量预计10,000台（占全球>50%），同比增长约200%；宇树、智元等机器人公司计划2026年IPO，估值约20亿欧元
- **整机结构**：典型人形机器人44个关节电机（Head/Arm/Body/Waist/Thigh/Calf/Ankle/Foot + 灵巧手）；深度相机、IMU、力矩传感器标配
- **关节模块挑战**：空间有限（多组件集成）；峰值功率需求为额定功率3倍且持续3秒；每台机器人>30个关节模块需一致性保证
- **电机驱动技术**：空心轴电机+圆形驱动板主流；手臂采用谐波减速器（100:1），腿部采用行星减速器（10:1）；双编码器检测电机与减速器速度/位置
- **英飞凌产品定位**：XENSIV 传感器（磁性/压力/雷达/ToF）、MOTIX 电机驱动 MCU、OptiMOS/CoolGaN MOSFET/HEMT、PSOC 主控、电源管理 IC
