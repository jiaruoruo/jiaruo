# Kimi K2（Moonshot AI）

## Description

Moonshot AI（月之暗面）是中国 AI 创业公司，以 Kimi 系列大语言模型著称。K2 系列是其旗舰推理模型，采用 MoE 架构（1T 参数规模），K2.6 是 2026 年发布的深度思考版本。Kimi 团队曾率先将 Muon 优化器（MuonClip 变种）应用于超大规模 MoE 训练（1T 参数，15.5T token，全程零崩溃），后被 DeepSeek V4 借鉴采用。

## Key Contributions

- **Muon 优化器工业级验证**：率先在 1T 参数 MoE 上验证 Muon（MuonClip 变种），全程零崩溃，被 DeepSeek V4 借鉴
- **SWE Pro 专业工程领先**：K2.6 在 SWE Pro（58.6）上超越所有西方旗舰模型，是国产模型工程实战能力的代表
- **HLE + 工具协同**：HLE w/ tools（54.0）全球第一，工具调用与推理协同能力出色
- **弱项明显**：Apex 推理仅 24.0，SimpleQA 事实问答 36.9，与顶级西方模型差距较大
