# 英飞凌科技（Infineon Technologies）

## Description

德国半导体公司，全球领先的功率半导体和微控制器供应商。在机器人领域提供覆盖电机驱动（MOSFET/GaN/GateDriver）、传感（XENSIV 系列：磁性/压力/雷达/ToF）、MCU（PSOC/AURIX/MOTIX）、电源管理的全系解决方案。中国人形机器人市场预计2025年出货量15K台（约全球70%份额），英飞凌作为半导体方案提供商积极布局。

## Key Contributions

- **XENSIV 传感器系列**：24/60GHz 雷达（障碍检测/手势识别）、REAL3 3D iToF（物体识别/导航）、MEMS 麦克风、3D 磁传感器（位置/碰撞检测）、气压传感器
- **电机驱动器件**：OptiMOS/StrongIRFET MOSFET；CoolGaN HEMTs（高效率/高功率密度，人形机器人差异化方向）；EiceDRIVER/MOTIX 门驱 IC
- **MCU 产品**：PSOC Control C3M（电机控制）、PSOC 4/6（HMI）、AURIX TC3/TC4（功能安全）、MOTIX MCU（电机驱动专用）、XMC 4000/7000（主控/HMI）
- **人形机器人方案**：完整 BOM 覆盖（电机控制+传感+电源+主控+通信），单台半导体 BOM 可寻址市场约$451（Infineon SAM）
