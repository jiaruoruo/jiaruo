# MiniMax

## Description

MiniMax 是一家中国 AI 公司，专注于开发多模态大模型及对外开放 API 平台。旗下模型覆盖文本生成（M 系列）、语音合成（speech 系列）、视频生成（Hailuo 系列）、图像生成（image 系列）和音乐生成（music 系列）等多个模态，提供按量付费和 Token Plan 两种商业模式。

## Key Contributions

- 推出 MiniMax M2 系列大语言模型，支持最大 204,800 token 上下文
- 提供兼容 Anthropic SDK 和 OpenAI SDK 的 API 接入方式，降低迁移成本
- 发布 Hailuo 视频生成模型系列（支持 1080P/10s）
- 提供官方 MCP Server（Python/JS 版本），将多模态能力接入 AI Agent 生态
