# 宇树机器人（Unitree Robotics）

## Description

中国四足/人形机器人领域代表企业，以强化学习技术、准直驱电机自研和开源生态著称。核心产品包括 Go2 四足机器人和 H1 人形机器人（TRL 7，180cm，47kg，27 DOF，行走速度3.3m/s）。开源策略吸引大量开发者，Unitree RL Gym（GitHub 1.8k stars）是最活跃的机器人 RL 训练环境之一。

## Key Contributions

- 强化学习步态控制：Isaac Gym 16,000+ 并行环境 + PPO，Sim-to-Real 成功率>90%，训练周期4-24小时
- 准直驱电机自研：减速比1:6，额定35Nm/峰值80Nm，力控带宽>100Hz，支持柔顺控制
- 开源生态贡献：Unitree RL Gym（官方 RL 训练环境）、Unitree Guide SDK（Python/C++）
- H1 人形机器人：行走速度3.3m/s（业界领先），模块化关节设计，售价约¥16-20万（估算）
