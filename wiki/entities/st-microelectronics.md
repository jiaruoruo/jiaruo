# 意法半导体（STMicroelectronics）

## Description

意法联合半导体公司，全球主要 MCU/MPU 供应商，STM32 系列是嵌入式领域最广泛使用的 MCU 之一。在机器人领域以 STM32 系列（N6/MP2/H7 等）为核心，覆盖可视化、处理控制、连接、传感、执行器驱动和电源六大模块。2026年3月发布 V9 版机器人解决方案，支持人形机器人完整方案。

## Key Contributions

- **STM32N6**：内置 NPU，视觉 AI + 运动控制，21手指关键点→15伺服电机控制 Demo，支持与 PLC 的 CANFD 通信
- **STM32MP257**：MPU，PLC 控制器，3路 Ethernet（含 EtherCAT 支持），RS-485/Modbus/CAN，隔离数字 I/O
- **灵巧手方案**：RA4T1 MCU（与瑞萨合作 Ruiyan）+ CAN-FD + Micro-ROS，5-15伺服电机控制，滚珠丝杠/触觉传感器
- **整机 BOM 估算能力**：系统级 BOM 分析（$17,900/台，44轴人形机器人），为客户提供完整方案参考
